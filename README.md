## Model

## 최종목표

## 고찰

데이터셋을 직접 만드는 과정에서 많은 더미 데이터와 분류가 정확하게 되어있지 않은 라벨 작업중인 데이터들을 직접 값을 입력해야 해주는 시간이 많이 걸렸으며
제공하는 한국어 모델들을 적용하여 사용하여 보았으면 조금 더 좋은 결과가 나오지 않았을까 생각합니다.
또한, 생각보다 단어중 한번만 나오고 마는 단어들이 많아 학습하기에 좋은 효과를 얻기 어려웠습니다.
